# HISTOFACT

HISTOFACT is an interactive web application built with Streamlit that allows users to explore historical facts, take quizzes, and learn about history in an engaging way.

## Features

- User authentication system
- Interactive historical facts dashboard
- Timeline visualization of historical events
- Search and explore historical facts by category, keyword, or time period
- Quiz system powered by Google's Gemini AI
- Favorites system to save interesting facts
