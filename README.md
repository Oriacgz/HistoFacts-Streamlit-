# HISTOFACTS - Historical Facts Application

HISTOFACTS is an interactive web application built with Streamlit that allows users to explore historical events, take quizzes, and learn about history in an engaging way.

## Features

- **User Authentication**: Sign up and login functionality
- **Historical Dashboard**: View historical events for any date
- **Timeline View**: Visualize historical events in chronological order
- **Search Functionality**: Search for specific historical events or topics
- **Quiz System**: Test your knowledge with quizzes on various historical topics
- **Favorites & Bookmarks**: Save interesting historical events for later reference
